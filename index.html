Welcome to Argene's portfolio site!

Argene is a software developer/programmer with several years experience in the game development industry. He already developed more than 20 games using Actionscript 3.0. To amplify his expertise in the programming field, he currently explores HTML, CSS, Javascript and React JS while continuously upskills to rapidly adapt to a morphing technological environment.
